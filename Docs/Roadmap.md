# Roadmap

## Milestone 1 — Playable Core Loop
- Implement deterministic board, piece motion, lock, clear, and fail rules.
- Add responsive touch controls and reliable input buffering.
- Ship a minimal playable loop with scoring and restart flow.
- Validate baseline performance on representative Android devices.

## Milestone 2 — Polished Controls and Feel
- Tune movement timing, DAS/ARR behavior, and gesture ergonomics.
- Add core feedback pass: impacts, clears, combo signals, and haptics.
- Improve session pacing and difficulty ramp consistency.
- Establish quality gates for responsiveness and readability.

## Milestone 3 — UX/UI Polish
- Build production-grade HUD, menus, pause, and game-over flows.
- Refine onboarding prompts and first-session clarity.
- Add settings for controls, audio, and accessibility-critical options.
- Harden visual hierarchy for diverse Android screen sizes.

## Milestone 4 — Content and Meta Systems
- Introduce progression scaffolding (profile stats, goals, unlock tracks).
- Add data-driven event/challenge framework for retention loops.
- Prepare economy/reward surfaces in a monetization-ready structure.
- Validate save robustness, versioning, and migration behavior.

## Milestone 5 — Release Preparation
- Final Android optimization and device compatibility verification.
- Integrate store-facing systems (billing/ads/consent) as approved.
- Complete telemetry, crash monitoring, and release candidate checks.
- Finalize Google Play listing assets, compliance, and launch checklist.
